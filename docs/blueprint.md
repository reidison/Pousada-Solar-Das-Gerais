# **App Name**: Solar Info Hub

## Core Features:

- Breakfast Info: Display breakfast hours and location.
- WiFi Details: Show the WiFi network name and password.
- Main Door Access: Display the access code for the main door.
- City Tour Booking: Show city tour schedule and booking instructions.
- Reception Contact: Provide a button to initiate a WhatsApp chat with reception.
- Minibar Menu: Display a menu of available items and prices.
- Welcome Message Customizer: AI-powered tool that adjusts welcome messages according to weather conditions. This is designed to use external information to add personalized flair.

## Style Guidelines:

- Primary color: Navy blue (#2E4057) to reflect the brand's sophisticated tone.
- Background color: Light off-white (#F9F6F2), providing a soft, clean base.
- Accent color: Muted gold (#A29373) for highlights, buttons, and decorative elements.
- Body and headline font: 'Poppins' for a contemporary, precise feel.
- Use Lucide icons to represent each information category. The icons should match the color scheme.
- Responsive design with cards or sections. Header remains fixed at the top. A 'back to top' button should appear when scrolling.
- Smooth transition animations when scrolling.
